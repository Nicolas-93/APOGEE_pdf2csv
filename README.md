# ApogeePDFExtractor

```shell
usage: APOGEE Extractor [-h] [-f FILE] [-o OUTPUT] [-i]

Extracts APOGEE data from PDF

optional arguments:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  PDF file
  -o OUTPUT, --output OUTPUT
                        output filename
  -i, --id              labels modules with their ID
```
